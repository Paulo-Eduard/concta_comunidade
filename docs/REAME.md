# Conecta Comunidade

Sistema de gerenciamento escolar para integração de alunos e professores, focado em cursos, quiz e suporte via chat.

## Funcionalidades
- **Autenticação:** Login e cadastro de alunos e professores.
- **Cursos:** Acesso a módulos de estudo com controle de progresso.
- **Quiz:** Avaliação de conhecimento com sistema de notas.
- **Chat:** Canal de comunicação direta entre alunos e professores.
- **Gestão (Professor):** Painel para listar e excluir alunos inativos.
- **Perfil:** Personalização de foto e dados do usuário.

## Como instalar
1. Clone o repositório: `git clone <url-do-seu-repo>`
2. Crie um ambiente virtual: `python -m venv .venv`
3. Ative o ambiente:
   - Windows: `.venv\Scripts\activate`
   - Linux/Mac: `source .venv/bin/activate`
4. Instale as dependências: `pip install -r requirements.txt`
5. Execute o sistema: `python main.py`
