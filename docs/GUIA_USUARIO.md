# Guia do Usuário

## Aluno
- **Cursos:** Acesse e marque módulos como concluídos.
- **Chat:** Envie suas dúvidas diretamente para o professor.
- **Perfil:** Mantenha sua foto de perfil atualizada.

## Professor
- **Gerenciamento:** No menu principal, utilize o botão "Gerenciar Alunos" para visualizar a lista de matriculados.
-Senha de Cadastrar: vendedores "1945".
- **Exclusão:** Utilize o botão "Excluir" com cautela; isso removerá o acesso do aluno e seu histórico de progresso do banco de dados.
