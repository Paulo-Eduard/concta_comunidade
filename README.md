# Conecta Comunidade 🌎💻

O **Conecta Comunidade** é um projeto desenvolvido em Python com foco em inclusão digital, comunicação e acessibilidade para a comunidade. O sistema foi criado para facilitar a interação entre usuários através de uma interface moderna, organizada e intuitiva.

---

# 🚀 Objetivo do Projeto

O projeto tem como objetivo criar uma plataforma capaz de:

* Conectar pessoas da comunidade
* Facilitar o acesso à informação
* Melhorar a inclusão digital
* Oferecer uma interface simples e moderna
* Integrar inteligência artificial futuramente
* Auxiliar usuários através de automações

---

# 🛠 Tecnologias Utilizadas

## Linguagem Principal

* Python

## Bibliotecas e Ferramentas

* CustomTkinter
* Tkinter
* SQLite / Banco de Dados
* JSON
* Pillow
* Ollama (IA local)
* VS Code

---

# 💻 Funcionalidades

* Sistema de Login
* Cadastro de Usuários
* Área de usuário
* Interface moderna com CustomTkinter
* Sistema de autenticação
* Banco de dados integrado
* Organização de informações da comunidade
* Estrutura preparada para IA local

# ▶ Como Executar o Projeto

## 1️⃣ Clonar o repositório

```bash
git clone https://github.com/Paulo-Eduard/conecta_comunidade.git
```

---

## 2️⃣ Entrar na pasta do projeto

```bash
cd conecta_comunidade
```

---

## 3️⃣ Criar ambiente virtual

```bash
python -m venv .venv
```

---

## 4️⃣ Ativar ambiente virtual

### Windows

```bash
.venv\Scripts\activate
```

---

## 5️⃣ Instalar dependências

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Executar o projeto

```bash
python main.py
```
